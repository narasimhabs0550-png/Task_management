# 🚀 Task Manager App

A full-stack Task Management application built with **React, Flask, and MySQL**.  
This app allows users to register, login, and manage their daily tasks efficiently.

---

## ✨ Features

- 🔐 User Authentication (JWT)
- 📝 Create, Read, Update, Delete Tasks (CRUD)
- 👤 User-specific task management
- 🔒 Secure password hashing (bcrypt)
- 📊 Dashboard UI
- ⚡ Fast and responsive frontend (React)

---

## 🛠 Tech Stack

**Frontend:**
- React.js
- Tailwind CSS

**Backend:**
- Python (Flask)
- Flask-JWT-Extended

**Database:**
- MySQL
